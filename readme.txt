Authorize.Net Payment Gateway for WooCommerce - Pay using Credit Card

== Description ==

= Introduction =
Do you want the fastest solution to accept Credit Cards? Authorize.net offer easy, safe, and affordable credit card processing.
Our Authorize.Net Payment Gateway for WooCommerce lets you accept credit card payments securely. With this plugin, your customer never have to leave your website for making payments.

= Features =
* Support Major Credit Cards & Debit Cards: Visa, MasterCard, American Express, Discover, JCB, Diner’s Club etc.
* Pay using Credit Card through Authorize.Net.
* Customers never leave your site during checkout.

== Installation ==

1. Upload the plugin folder to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Thats it – you can now configure the plugin.
